## Project description ##
SomaShare Is an online web application that aids in the selling and purchasing of tertiary educational materials between student communities. Users can make ads, buy and sell textbooks as well as review their experience on the web application.

## Installation ##

Option 1
1. Open up Visual Studio Code 2026
2. Click on 'Clone a Repository'
3. Enter this URL into the location
https://github.com/Skypie14/SomaShare.git

## OR ##

Option 2
1. Download the ZIP file connected to the Assignment.
2. Unzip the file
3. Open up Visual Studio Code 2026
4. Click on 'Open a folder'
5. Find the located assignment [25301714_25302591_SS2]
6. open on the file Soma Share

## THEN ##

1. Delete the Migration file
2. Open up appsettings.json
3. Change the ConnectionString to your database
4. If using Microsoft SQL Server Management, you find the server name and replace the words after Server = [eg: Server= "YourServerName//SQLEXPRESS;Database=YourDatabase;Trusted_Connection=True;TrustServerCertificate=True;"
5. Open up NU Get manager then Package Manager Console
6. In the console type "Add-migration Intial"
7. Once running, then type "update-database"

 Please note to not double click on buttons.
